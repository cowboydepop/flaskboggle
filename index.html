Boggle Game

How to Play

  • Starting the Game: Click the "Start" button to begin. A grid of random letters will appear.
  • Finding Words: Form words by connecting letters that are next to each other in any direction - up, down, left, right, or diagonally. You can't use the same letter twice in one word.
  • End of the Game: When the timer reaches zero, the game ends. Your score will be displayed along with the high score and the number of games played. You can start a new game by pressing the "Start" button again.
